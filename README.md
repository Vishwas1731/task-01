# task-01
A Python Pandas project demonstrating data cleaning on the Netflix Movies and TV Shows dataset. It handles missing values, removes duplicates, standardizes text fields, fixes date formats, renames columns, and prepares clean data for analysis and ML tasks.

# Task 1: Data Cleaning and Preparation (Netflix Dataset)

## Objective
To clean and preprocess a raw dataset with missing values, duplicates, and inconsistent formats using Python (Pandas).

## Dataset
Netflix Movies and TV Shows dataset from Kaggle.

## Tools Used
- Python
- Pandas
- colab 

## Steps Performed
- Identified and handled missing values
- Removed duplicate rows
- Standardized text values
- Converted date formats
- Renamed columns for consistency
- Fixed data types

## Files in this Repository
- dataset/netflix_titles.csv – raw dataset
- code/data_cleaning.ipynb – cleaning code
- screenshots/ – output images 

## Output
A cleaned dataset ready for analysis.
